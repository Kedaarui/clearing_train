# 🏦 自动清账系统 (Automatic Clearing System)

基于深度学习的财务凭证自动清账系统，使用图神经网络和组合优化算法自动识别可清账的凭证组合。


## 🎯 项目简介

该系统旨在自动化财务清账流程，通过深度学习模型学习凭证之间的清账规则，无需人工定义复杂的业务规则。

### 业务场景

在财务系统中，需要定期清理应付/应收账款。清账的核心规则是：
- 找出一组凭证，使得它们的金额相加为 0
- 这些凭证需要满足特定的业务约束（如：相同的公司代码、利润中心等）

传统方法需要财务人员手动定义规则，而本系统可以**自动学习**这些规则。

## 🧠 算法原理

### 两阶段方法

阶段 1: 深度学习 - 学习凭证兼容性
  输入: 两个凭证的特征
  输出: 兼容性得分 (0-1)
  作用: 过滤不可能的组合，减少搜索空间
阶段 2: 图搜索 + 组合优化

构建兼容图（节点=凭证，边=兼容性>阈值）
分解为连通分量
在每个分量中搜索零和子集
去重优化


### 模型架构

输入层: 凭证特征 (Company Code, Account, Profit Center, etc.)
   ↓
编码器: 全连接神经网络 (64→32→16)
   ↓
兼容性判断: 成对特征拼接 + 分类器
   ↓
输出: 兼容性概率
